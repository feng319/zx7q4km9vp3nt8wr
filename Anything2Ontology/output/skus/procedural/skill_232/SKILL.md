# 概述
基于永辉超市成熟的合伙人+赛马机制设计无实股全覆盖的门店激励方案，绑定全员利益
# 步骤
1. 核心机制采用合伙人+赛马机制，不设置全员股权分配
2. 搭建三级奖金包体系：个人奖包、门店奖包、集体奖包
3. 确定核心激励对象范围：仅将店长、店助列为可持股对象，无需覆盖课长、经理层级
4. 业绩核算规则：依据内部赛马机制（费尔德新筹法）排列毛利率，综合全员意见确定最终业绩排名
5. 明确利润分配规则：清晰划分存量利润与增量利润的分配方式，实现全员绑定指标、利益关联
# 预期结果
有效降低门店运营损耗（例如果蔬品类损耗率可降至5%，远低于行业平均水平），损耗减少直接转化为利润提升
