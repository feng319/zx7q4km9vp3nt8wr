**一、餐饮**

**1.贤合庄**

贤合庄由陈赫、叶一茜、朱桢等二三线明星联合创立。**明星效应在港台餐饮行业已趋饱和，现转向大陆市场发展**。优势资本团队与吕良伟夫妇、梁朝伟夫妇、甄子丹夫

妇、张智霖夫妇等明星保持合作关系，共同参与一带一路相关赞助项目。

**2.资本与明星**

**明星品牌流量变现需依赖资本支持**。传统影视收入占比从80%-90%下降至20%，未来市场收益将主要来源于其他渠道。**餐饮行业成为明星流量变现的重要途径** ，但并非所有明星餐饮项目都能成功（如王菲上海酒吧经营不善，但其早期餐厅表现尚可）。贤合庄能否成为下一个海底捞仍存不确定性。

**3.加盟盈利模式**

**餐饮连锁盈利模式差异分析** ：

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **模式类型** | **资金投入方** | **运营主体** | **估值影响** | **典型案例** |
| 直营店 | 母公司 | 母公司 | 高 | - |
| 加盟店 | 加盟商 | 加盟商 | 低 | 周六福（3400家店中仅28家直营） |
| 托管店 | 合作伙伴 | 母公司 | 较高 | - |

**核心观点** ：

l **加盟费是品牌方重要盈利来源** ，但单一依赖加盟费会导致估值偏低

l **有效商业模式需确保加盟商2年内回本**

l **上市企业需平衡托管店与加盟店比例（建议1:1）** ，纯加盟模式难以通过上市审核（如周六福IPO受阻）

l **托管模式优势** ：转移租金、人力成本至合作伙伴，抗风险能力更强（对比疫情期间直营火锅/珠宝/美容企业大规模倒闭）

**4.资本结构**

贤合庄采用**典型有限合伙架构** ：

l **股权设计** ：三位明星作为LP（陈赫为大股东），GP由普通合伙人担任

l **资本运作** ：通过有限合伙企业控股运营公司

l **扩张策略** ：**每个省份设立直营店作为基地，配套发展托管店 - 资本价值关键** ：**托管店占比需达50%以上** ，纯加盟模式资本市场认可度低

l **成本管理原则** ：**通过模式设计转移成本而非削减成本** ，避免品质下降（如托管模式将经营成本转移至合作伙伴）

**一、企业转移成本**

企业应避免盲目降低成本，而应通过成本转移策略优化商业模式。将成本转移至上下游合作伙伴，可维持产品品质并构建新型商业生态。

**二、百果园商业模式分析**

**1.百果园的门店和销售额**

百果园全国门店数量约4100家，2019年销售额达120亿元，2020年略有下滑但保持稳定。

**2.百果园的合作伙伴**

百果园采用**平台加合伙人**模式，其核心竞争力在于**全产业链控制**（涵盖上游种植、下游销售及自主物流），并通过**产地直发**降低损耗率。

**3.百果园的价值销售渠道**

百果园的核心价值在于**全覆盖的销售渠道** ，但其生鲜商业模式难以复制，主要因行业竞争激烈且运营复杂度高。

**4.生鲜模式经营难**

生鲜行业经营难点包括：

l **多维度竞争** ：超市、便利店、电商等多渠道分流

l **高损耗率** ：产品易腐、窗口期短、人力成本高

l **利润率矛盾** ：毛利率高但净利润率低，传统加盟模式难以创造显著企业价值

**5.百果园的三种模式**

百果园渠道运营分为三类：

l **传统加盟模式** ：收取加盟费

l **共同投资模式** ：百果园控股51%，合作方持股49%

l **合伙人托管模式** ：合作方出资，百果园全权管理

共同投资模式下子公司注册资本需全额实缴，但**合伙人模式估值潜力最大**。

**6.合伙人模式**

**合伙人托管模式**允许企业/个人出资开店，由百果园统一管理，并可通过**客户众筹**进一步拓展资金渠道。

**7.百果园的营业额**

百果园年营业额超百亿元，规模超越海底捞等企业。其未来可借鉴**瑞幸咖啡的互联网化路径** ，通过数字化转型提升估值（市盈率可从50倍增至200倍）。

**一、贝壳**

贝壳找房上市前估值为140亿元，上市后获得巨额融资。优势资本曾投资房地产中介企业易居中国并成功上市，但贝壳找房的核心价值在于其门店网络。

**- 贝壳创始人背景**

链家董事长左辉曾进入失信人名单，但通过剥离贝壳业务实现重新上市。链家在全国拥有8000家门店和12万经纪人，而贝壳拥有42000家门店和45万经纪人，人均年成交3-4套房产。贝壳平台的成交额是链家的1倍以上，其平台模式具有快速扩张能力。

**- 58同城盈利模式**

58同城通过整合低频服务（如房产、招聘、家政）形成高频流量平台，主要盈利模式为收取中介端口费及成交分佣。2020年58同城净利润约79亿元，而贝壳找房亏损21亿元。贝壳虽亏损但股权价值显著，核心在于其45万经纪人的规模效应。

**- 贝壳成功因素**

贝壳的成功源于平台化赋能策略，通过标准化非标房产中介服务实现差异化竞争。未来可能细分为一二线城市、三四线城市、别墅专营等垂直领域贝壳，针对高净值客户拓展衍生收益。

**- 房地产中介企业命运**

独立房地产中介企业难以长期存活，必须选择成为上市公司或被上市公司并购。

**- 应用案例**

1) 例题:三四线城市贝壳定位

企业百纳航计划定位为三四线城市贝壳，联合中小中介共同入股，通过商业模式设计实现资本化。当前房产中介核心痛点在于一手房销售需缴纳高额保证金（如2000万- 4000万），解决方案是通过融资覆盖保证金成本。优势资本正与地方政府及上海同策合作产业勾地项目，但需解决房产去化难题。

**- 未来趋势**

未来将涌现更多细分领域贝壳模式。若无法成为贝壳或入股贝壳，可创造新细分品

类。贝壳的研发投入是58同城的3倍，其核心竞争力包括VR看房、大数据筛选等技

术，以及"真房源"承诺的差异化服务。房产经纪人年均成交3-4套，短视频平台虽能提升销量但存在流量限制。

**一、云集微店的商业模式**

**1.七大电商概述**

**2.云集微店的核心竞争力**

**云集微店的核心竞争力在于社交与交易的双重整合** ，成为国内首个实现社交化交易的电商平台。

**3.云集微店的发展历程**

云集微店2015年上线，主营**美妆**、**数码**、**母婴**、**生鲜**等品类。获得鼎晖投资、中鼎资本注资后，平台曾拥有80万微商代理，后因涉嫌传销被杭州政府处罚300万元。

**4.云集微店的商业模式转型**

云集微店从微商模式转型为**S2B2C模式** ，其裂变机制基于指数级增长模型（2 ⁿ )。国内微商主要集聚于**杭州**和**广州** ，云集微店总部位于杭州。

**5.云集微店的供应链与分销模式**

转型后供应链结构为：

l **上游** ：品牌方、经销商、生产厂商

l **下游** ：店长与终端消费者

分销模式通过店主向消费者完成，上市时定位为**中国会员电商第一股** ，但仍保留微商特征。

**6.云集微店合规化**

合规化关键措施：

l **协议主体** ：采用个体户或公司签约替代个人签约

l **层级管理** ：明确一级至四级经销商体系

此模式规避了传销风险，成为社交电商合规范本。

**7.适合做微商的品类**

**高毛利、高频复购品类**最适合微商模式：

l **化妆品/护肤品**

l **休闲食品**

l **红酒**

l **教育产品**

典型案例：向日葵妈妈年营收5亿，专注语文/英语等教育领域。

**8.云集微店会员模式**

会员体系分为：

l **VIP会员** ：缴纳约400元入门费

l **钻石会员** ：升级后享有更高权益

激励机制：店主邀请新会员购物可获得返利，下级店长业绩关联上级收益。

**一、自我介绍与阿里巴巴金融改革举例**

l **银行借贷流程与阿里巴巴金融效率对比** ：银行借贷无论金额大小流程相同，而阿里巴巴平台老客户贷款发放仅需3小时，新客户仅需3天，体现自下而上的金融改革效率。

l **2014年阿里巴巴金融业务受阻事件** ：上市前余额宝与二维码支付业务被叫停，因触及传统银行活期存款利益及银联支付垄断地位，但最终仍获准上市。

l **马云投资恒大足球队的舆论影响** ：12亿投资导致股价暴跌，马云公开回应称参与足球领域是为解决问题，却反被舆论质疑，类似冯小刚导演参与春晚的争议。

l **体制内外视角差异** ：未经历体制者易抱怨“市场经济计划权利”，而兼具体制内外经验者可能形成不同结论，阿里巴巴的发展模式印证此观点。

**二、全产业链模式**

l **全产业链金融整合必要性** ：未来产业链需结合贷款融资功能，**蚂蚁金服**为淘宝商家提供销售贷款，**可口可乐**为生产商提供生产贷款，**蒙牛**联合农业银行为农民提供养殖贷款。

l **广东温氏集团案例** ：通过金融支持五万农户自主选择直接购买或贷款获取养殖资源（鸡、牛、猪），体现全产业链金融的终端渗透。

l **金融平台建设目标** ：全产业链模式最终需建立**金融租赁公司** ，形成**资金运作**、**资产运作**、**资本运作**三位一体的综合平台。
