## 概述
锌铁液流电池的功率部件（电堆）与容量部件（电解液储罐）物理分离，可独立扩容。本技能说明如何通过增加储能时长降低单位容量的装机成本，适用于项目设计阶段的技术经济优化。

## 步骤
1. **确定基础设计方案**
   - 选定基础方案：如100kW/400kWh（4小时储能时长）
   - 记录容量成本（元/kWh）和功率成本（元/kW），以及对应的放电效率。

2. **计算基础方案装机成本**
   - 装机成本0 = (容量成本 / 放电效率) + (功率成本 / 初始放电时长)

3. **设定扩容方案**
   - 保持功率部件不变（同一套100kW电堆），仅增加储罐容量至800kWh（8小时时长）。
   - 扩容后的放电时长 = 新容量 / 原功率 = 800/100 = 8h

4. **计算扩容后装机成本**
   - 装机成本1 = (容量成本 / 放电效率) + (功率成本 / 新放电时长)
   - 与基础方案相比，容量部分不变，功率分摊金额下降。

5. **比较成本节约**
   - 成本降低比例 = (装机成本0 - 装机成本1) / 装机成本0 × 100%
   - 同时需考虑增加储罐带来的占地、基础建设等附加成本，需在预算中列入。

6. **综合考虑全生命周期经济性**
   - 使用LCOS计算方法评估新方案的度电成本，确认因循环寿命长、运维成本随容量增加有限而带来的总体经济性改善。

## 决策点
- **如果场地充裕且增加储罐的边际成本较低**：推荐采用长时储能方案（6～10小时）。
- **如果功率设备成本占比很高**：延长时长对降本效果更显著，值得优先考虑。
- **如果项目对放电倍率有特殊要求**：需权衡功率部件是否兼容更长时放电（如电堆电流密度），一般液流电池可灵活适配。

## 预期结果
得到优化后的装机成本及成本降幅，为项目投资决策提供“通过增加储能时长降低初始投资”的量化依据，同时可结合LCOS分析最终锁定最优储能时长配置。
