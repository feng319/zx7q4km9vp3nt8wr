# 偏差内部结算流程
## 概述
在独立结算原则下，处理场站间出力互补产生的代发费用结算，符合审计、合规要求
## 步骤
1. 核对数据：确认各场站计划发电量、实际发电量，计算各场站偏差值
2. 确认整体完成情况：核实集团整体中标电量是否达标
3. 核算代发费用：按照预先制定的内部影子价格体系，核算欠发场站需向超发场站支付的代发补偿费
4. 完成结算：执行内部资金划转，出具符合审计要求的结算凭证
## 决策点
- 如果集团整体未完成中标电量：先按市场规则缴纳偏差罚款，再按内部规则将罚款分摊至对应责任场站
## 预期结果
内部结算清晰合规，无股东纠纷，符合税务、审计要求
