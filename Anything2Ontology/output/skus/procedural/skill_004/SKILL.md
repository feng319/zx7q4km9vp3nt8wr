# 中标发电计划分解流程
## 概述
合理分配集中中标电量至各物理场站，避免违约，符合电网运行要求
## 步骤
1. 收集基础数据：各场站实时出力、超短期功率预测结果、所在输电断面阻塞情况
2. 选择分配算法：根据管理目标匹配合适的分配算法完成初步分配
3. 对接电网调度：确认分配结果符合电网物理运行约束
4. 下发计划：将最终发电计划下达至各场站执行
## 决策点
- 如果优先保障分配公平：采用按容量比例分配算法
- 如果优先降低整体违约风险：采用按预测能力分配算法
- 如果优先符合电网物理约束：采用按网络约束（SCED）分配算法
## 预期结果
各场站获得明确发电计划，整体中标电量完成率达标，无违约
