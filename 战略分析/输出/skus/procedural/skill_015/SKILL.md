# 概述
通过企业生命周期、产业生命周期、资本周期三个周期的叠加效应，判断企业当前所处的发展位置和风险机遇。

# 步骤
1. 分别判断三个周期的位置：
   - 企业生命周期：对应创业/成长/盛年/衰退四阶段
   - 产业周期：根据供需变化、技术成熟度判断所处阶段
   - 资本周期：根据折现率、资本冷热判断资本周期位置
2. 分析三个周期的叠加效应

# 决策点
- 如果三个周期波峰叠加：加大投入扩大优势，最大化企业价值
- 如果三个周期波谷叠加：收缩现金流，做好风险应对，避免快速死亡

# 预期结果
准确识别企业当前发展状态，提前应对风险，抓住发展机遇
