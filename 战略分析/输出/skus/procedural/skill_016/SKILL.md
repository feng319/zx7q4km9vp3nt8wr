# 概述
将企业生命周期划分为创业、成长、盛年、衰退四个阶段，针对不同阶段的核心特征匹配对应的应对策略。

# 步骤
1. 诊断企业所处生命周期阶段：
   - 创业阶段：现金流紧张，创始人承诺和行动为核心驱动力
   - 成长阶段：销售导向，初步搭建制度体系，存在大量服务/产品不完善问题
   - 盛年阶段：产品成熟，客户复购率高，员工和客户规模持续扩大
   - 衰退阶段：形式大于功能，创新精神不足，创始人心力不足
2. 针对性采取对应应对措施

# 决策点
- 处于创业阶段：做好最差现金流预期，避免现金流断裂
- 处于成长阶段：提升战略优先级思考能力，避免产品/体系不成熟时过度扩张
- 处于盛年阶段：建立成熟组织管理制度，平衡授权和分权的关系
- 处于衰退阶段：孵化新业务，激发组织创新活力

# 预期结果
精准识别企业所处阶段，解决核心问题，延缓衰退，延长企业生命周期
