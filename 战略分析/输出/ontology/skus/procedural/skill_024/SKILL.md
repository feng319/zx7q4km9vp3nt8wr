# 概述
基于哈佛商学院颠覆式创新框架，通过三种策略布局创新业务，平衡现有业务和创新业务的发展。

# 步骤
根据创新业务特性、与主业协同度、企业资源情况选择对应策略：
1. Buy（投资/并购）：适合需要快速获取核心技术、完善产业链布局的创新方向
2. Borrow（外部孵化/合作）：适合风险较高、需要外部资源协同的创新方向
3. Build（内部创业/孵化）：适合和主业协同性强、内部团队能力匹配的创新方向

# 决策点
- 创新业务和主业协同性低、技术成熟度低：优先选择Borrow策略
- 创新业务属于产业链核心环节、需要快速获取：优先选择Buy策略
- 创新业务和主业协同性高、团队能力匹配：优先选择Build策略

# 预期结果
高效布局创新业务，避免被颠覆式创新冲击，实现持续增长
