# 集中报价风控体系搭建流程
## 概述
对冲现货市场价格波动风险，防范合规风险，稳定集团收益
## 步骤
1. 构建资产组合模型：整合集团内风光资产数据，建立资产组合分析框架
2. 搭建风险量化模型：针对极端高价、负电价场景，建立基于组合资产的风险价值（VaR）模型
3. 制定策略库：设定不同场景下的止损线和报价策略库
4. 建立信息隔离机制：实现发电侧报价团队与售电侧团队的物理、信息隔离，设置IT系统权限管控
5. 定期审计：定期开展隔离有效性审计，留存IP地址、通话记录等审计证据
## 决策点
- 如果遇到极端负电价时段：可策略性关停部分低效机组，通过适度物理持留推高价格，保障整体收益
## 预期结果
有效对冲市场价格波动风险，无内幕交易合规风险，整体收益稳定
