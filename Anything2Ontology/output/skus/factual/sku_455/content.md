### 核心特征
裂变思路与微商模式相似

### 适用人员
销售冠军、区域总代、区域销售总经理等群体听完相关课程后，可选择进入资本界，或参与内部创业获得资金支持

### 创业申报流程
1. 公司推出高毛利新能源产品，有新项目时符合内部裂变条件的员工可向公司提交申报
2. 公司组织创业大赛评审申报项目，若项目可行则组建新公司独立运作
3. 新公司可由总经理（多为销售冠军）联合管理层、技术总监组建团队并引入外部资本
4. 项目需获得所有相关方认可（而非仅老板认可），相关参与人员方可获得收益分配权和优先分红权

### 新公司股权分配规则
| 持股主体 | 持股比例 | 备注 |
| --- | --- | --- |
| 母公司创始人（综艺、张力） | 50% | 二人合计持有，非母公司主体持有 |
| 新公司总经理 | 10% | - |
| 新公司核心团队 | 15% | - |
| 参与投票的员工 | 25% | - |

### 分红规则
1. 符合条件的持股方享有优先分红权
2. 收益分配分为劳动分红、管理分红两类，劳动分红优先分配，管理分红后续分配
3. 机制配套设置内部虚拟股权相关规则
