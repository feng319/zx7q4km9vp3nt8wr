# 概述
本技能用于股权激励场景下，解决母公司可授予股份不足的问题，支持股东同时持有多层级公司股份并行使管理权。
# 步骤
1. 评估母公司用于股权激励的股份存量，确认是否存在股份减少、额度不足的情况
2. 确认公司允许母公司股东持有子公司股份的规则
3. 为相关人员配置持股身份：可同时作为母公司小股东、子公司大股东
4. 授予相关人员对应多个子公司的管理权限
# 决策点
- 如果股权激励场景下母公司股份减少/额度不足：可选择在子公司为相关人员授予股份
# 预期结果
完成多层级持股配置，相关人员可同时持有母公司、多个子公司股份，并行使对应子公司的管理权限
