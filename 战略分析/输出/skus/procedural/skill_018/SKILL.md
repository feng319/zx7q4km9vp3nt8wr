# 概述
平衡计分卡（BSC）是从财务、客户、内部运营、学习与成长四个维度，将组织战略落地为可操作衡量指标的战略管理工具。

# 步骤
1. 培训企业高层管理者，使其理解BSC的战略管理价值，CEO作为核心驱动者
2. 组建专职BSC项目推广团队，作为高层和项目之间的桥梁
3. 重新审视、明确企业的中长期战略目标，向全员传递战略方向
4. 结合企业发展阶段、竞争环境和行业特征，灵活确定BSC的评估维度（默认四个基础维度）
5. 为每个BSC维度选定可落地、可考核的KPI，责任落实到具体员工
6. 明确每个KPI的具体量化数值目标
7. 建立「just do it」的执行文化，开始落地推行

# 预期结果
实现财务与非财务指标、长期与短期目标、结果与动因指标的平衡，对齐各部门目标和公司战略，保障战略落地
