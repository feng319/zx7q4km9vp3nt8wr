# 概述
通过10个核心问题快速梳理商业模式的核心逻辑，判断项目的商业可行性。

# 步骤
依次回答以下10个问题：
1. 我/团队的核心特质和优势是什么，是否匹配当前创业方向
2. 项目所处的行业是什么，行业规模、增速、集中度如何
3. 选择行业价值链中的哪个价值点切入
4. 可以借助哪些外力（政策、技术、用户变化等）放大优势
5. 产品/服务的目标用户是谁
6. 为目标用户解决什么具体需求/痛点
7. 提供什么具体的产品与服务
8. 为用户实现什么可衡量的效果/价值
9. 达成阶段性目标需要多长时间
10. 构建什么核心壁垒避免竞争对手模仿

# 预期结果
能够1-2句话说清项目核心商业模式，明确项目的核心竞争力和商业价值
