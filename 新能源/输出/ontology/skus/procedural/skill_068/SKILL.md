## 概述
本技能提供锌铁液流电池储能系统全生命周期度电成本（LCOS）的标准化计算方法，涵盖装机成本、运维成本、电力损耗成本及发电量现值的折算，适用于100kW/400kWh等规模的储能电站投资评估。

## 所需参数
- 系统规模：额定功率（kW）、储能容量（kWh）
- 成本数据：容量成本（元/kWh）、功率成本（元/kW）
- 性能参数：放电效率、循环效率（交流侧，含自耗电）、年均循环次数
- 财务参数：折现率、运维年限、企业所得税率、股权成本、债务成本
- 市场参数：平均购电价格（元/kWh）

## 步骤
1. **确定基本规模与额定放电时长**
   - 储能容量＝额定功率 × 放电时长（h）
   - 例如：100kW/400kWh系统，放电时长为400/100＝4小时。

2. **计算初始装机成本（元/kWh）**
   装机成本 = (容量成本 / 放电效率) + (功率成本 / 放电时长)
   - 容量成本除以放电效率：因系统放电过程中有能量损耗，为放出1kWh需存储多于1kWh电量，需对应增加容量投资。
   - 功率成本除以放电时长：将单位功率成本分摊至每单位储能容量。

3. **计算运维成本现值（元/kWh）**
   - 年运维成本 = 装机成本 × 年运维费率（通常取3%～5%）
   - 运维成本现值 = ∑[年运维成本 / (1 + r)^t]，t从1到运维年限
   - r为折现率：若采用股权+债务模式，建议使用加权平均资本成本（WACC）：
     - r = (E/V)×Re + (D/V)×Rd×(1 - T)
     - E/V：自有资金比例，Re：股权成本
     - D/V：债务资金比例，Rd：债务成本（贷款利率）
     - T：企业所得税率

4. **计算电力损耗成本现值（元/kWh）**
   - 单次循环损耗电量 = (1/循环效率) - 1  kWh（每放出1kWh的充电损失）
   - 年损耗成本 = 年循环次数 × 单次损耗电量 × 平均购电价格
   - 电力损耗成本现值 = ∑[年损耗成本 / (1 + r)^t]，t从1到运维年限

5. **计算全生命周期发电量现值（kWh）**
   - 年放电量 = 系统储能容量 × 年循环次数（假设无衰减）
   - 发电量现值 = ∑[年放电量 / (1 + r)^t]，t从1到运维年限
   - 注：锌铁液流电池可通过电解液补充维护保持容量，可假设循环寿命内无显著衰减；若保守考虑，可采用首年衰减率或容量维持率调整。

6. **计算LCOS**
   LCOS = (装机成本 + 运维成本现值 + 电力损耗成本现值) / 发电量现值
   - 结果单位为元/kWh。

## 决策点
- **如果项目采用全股权融资**：折现率可直接使用股权成本Re。
- **如果项目采用多种融资方式**：采用WACC作为折现率，需根据实际资本结构计算。
- **如果预期容量存在衰减**：在每年发电量计算中引入衰减因子（如0.5%/年），液流电池通常可忽略或取极小值。

## 预期结果
输出一个单一数值，反映在该套参数下储能系统每提供1kWh电能的综合成本。可用于横向比较锂电池及其他储能技术的经济性，或评估项目可行性。
