# 概述
通过「定义战场-选择增长方式」两步法，帮助企业在周期波动中找到可持续的增长路径。

# 步骤
1. 定义战场：结合技术、市场、政策趋势重新定义业务边界，区分两类创新机会：
   - 低垂果实/青苹果：当前可见的、竞争充分的创新机会
   - 远处火山：3-5年后才会成熟的、存在创新海沟的长期机会
2. 选择增长方式：
   - 经营性增长：对应低垂果实机会，通过现有客户/区域做深、产品做广、技术迭代做透实现增长
   - 结构性增长：对应长期机会，通过技术迭代、新市场新人群、政策红利等实现突破性增长

# 落地方式
可通过共创战略、共同孵化、共同投资三种方式落地结构性增长项目

# 预期结果
找到适配企业的增长路径，穿越周期实现持续增长
