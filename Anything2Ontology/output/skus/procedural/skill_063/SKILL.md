# 概述
短流程炼钢以废钢为核心原料，无需烧结、焦化、高炉炼铁环节，能耗和碳排放远低于长流程炼钢，是行业降碳的重要路径。
# 步骤
1. 废钢熔化：将废钢置于电弧炉内，利用石墨电极产生的3000℃以上电弧将废钢熔化为钢水
2. 成分调整：调整钢水杂质、合金元素含量至符合产品要求
3. 连铸：将高温钢水连续浇铸到结晶器冷却凝固，制成板坯、方坯等固态半成品钢坯
4. 轧钢：对钢坯施加压力产生塑性变形，制成特定形状、尺寸、性能的成品钢材
# 预期结果
完成低排放炼钢生产，吨钢能耗约0.3吨标准煤，二氧化碳排放约1吨
