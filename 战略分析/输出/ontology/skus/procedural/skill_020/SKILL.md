# 概述
基于产业需求侧和供给侧特征，将产业发展分为天使市场、高增长市场、规模稳定市场三个阶段，匹配对应商业模式设计策略。

# 步骤
1. 判断所处产业发展阶段
2. 执行对应阶段的商业模式设计策略

# 决策点
- 如果属于天使市场（用户需求不明确、产业链未形成）：需求侧实施「标杆-出圈」策略，供给侧实施「共创-共建」策略
- 如果属于高增长市场（需求快速增长、产业链基本形成）：需求侧实施「占位-心智」策略，供给侧实施「借势-锁定」策略
- 如果属于规模稳定市场（需求稳定、产业链成熟）：需求侧实施「覆盖-终身」策略，供给侧实施「融合-升维」策略

# 预期结果
设计出适配当前产业阶段的商业模式，获取差异化竞争优势，突破增长瓶颈
