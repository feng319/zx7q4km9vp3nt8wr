# 概述
长流程炼钢是我国目前主流的钢铁生产工艺，以铁矿石和煤炭为核心原料，全流程能耗和碳排放较高。
# 步骤
1. 原料准备：分别完成铁矿石预处理（烧结/球团）、煤炭预处理（焦化，隔绝空气高温蒸馏制成焦炭）
2. 高炉炼铁：高温下利用焦炭作为还原剂，将铁矿石还原为纯金属生铁，去除杂质
3. 转炉炼钢：吹入氧气调整铁水碳、杂质元素含量，按需添加合金得到符合成分要求的钢水
4. 连铸：将高温钢水连续浇铸到结晶器冷却凝固，制成板坯、方坯等固态半成品钢坯
5. 轧钢：对钢坯施加压力产生塑性变形，制成特定形状、尺寸、性能的成品钢材
# 决策点
- 铁矿石预处理环节：
  - 如果使用普通铁矿石粉：采用烧结工艺，将铁粉与熔剂、燃料混合高温加热制成烧结矿
  - 如果使用细度均匀的高品位铁矿石精粉：采用球团工艺，制成1-2cm小球颗粒，能耗比烧结低20%-40%
- 轧钢环节：
  - 如果对钢材精度要求较低：采用热轧工艺，将钢材加热到1000℃以上轧制，能耗更低
  - 如果对钢材精度要求较高：采用冷轧工艺，室温轧制精度更高但能耗更高
# 预期结果
完成从原料到成品钢材的全流程生产，吨钢能耗约0.55-0.6吨标准煤，二氧化碳排放约2.2吨
