# 概述
本模式为流通医药行业乡村药房托管加盟运营方案，对标海澜之家托管逻辑，核心是打造县域龙头药店托管服务能力，实现低门槛标准化药房扩张。
# 步骤
1. 拓展加盟投资方，仅要求投资方完成出资，无需参与药房日常运营
2. 由品牌总部统一承接所有托管药房的运营管理权限
3. 重点搭建药房店长人才标准化培养体系，保障运营能力输出
# 预期结果
依托标准化托管运营体系，仅需普通资质的药房店长即可实现一流经营业绩，可快速扩张县域乡村药房网络。
