**一、抖音认知**

**1.抖音平台本质**

抖音平台本质并非高端商务场景，而是大众化市场。在抖音平台上，即使知名创业者在官方支持下，其影响力也难以超越接地气的摆摊创业或成功学内容。**核心逻辑在于用户需求多元化** ，大部分用户面临生活压力，**解压需求远大于增压需求**。平台6亿日活用户中，**95%仍处于温饱线生活水平**。

**2.官媒运营策略**

官媒运营同样遵循该逻辑，表现为**内容娱乐化转型** ，例如在网红账号评论区互动，甚至军事类视频配用搞笑背景音乐。

**3.内容创作原则**

抖音内容创作需平衡**人设塑造**与**流量获取** ，但二者兼得难度较大。**交替发布两类内容**是有效策略。

1) 精准度与粉丝量

**内容精准度虽重要，但缺乏背书的普通创作者难以实现指数级增长**。**粉丝量是核心背书指标** ，应在保证基础精准度的前提下优先提升粉丝规模。

**二、账号运营策略**

**粉丝量突破1万即可超越99%普通用户** ，该指标对求职、经商、投资等场景均具有显著杠杆效应。

**1.内容矩阵案例**

1) 专业内容定位

以硬科技、数字经济、新能源等专业领域为例，**专业内容虽流量数据不佳，但属于必要的人设建设素材** ，主要面向创业者及地方政府等专业客户群体，用于建立商业能力认可度。

2) 流量内容定位

金融类娱乐内容单条播放量可达100-1000万，**核心作用在于建立用户信任并降低交易成本**。具体表现为：创业者更易接受投资条款，政府机构更倾向选择非头部服务商。

**三、账号定位方法**

**账号定位需与商业目标严格对应** ：

l 求职类账号需突出工作能力证明

l 商品销售类需强化产品展示

l 招商加盟类需呈现客户成功案例

l 知识付费类需塑造专业权威形象

**一、创业实战打法**

**1.抖音内容剧情化**

l **情绪真实是视频制作的核心** ，可降低制作成本并保证可持续性。

l **内容剧情化的关键要素**包括：

o **一秒留人** ：通过冲突点前置或标题党措辞吸引用户。

o **五秒完播** ：延续冲突点以扩大事态，确保用户观看时长。

o **叙述内容** ：核心内容后置以延长用户停留时间。

o **植入槽点** ：增强互动性。

o **金句升华** ：通过引经据典或引起共鸣的押韵语句提升点赞收藏数据。

l **抖音推送机制依赖用户行为数据** ，新账号需重点优化一秒留人和互动数据。

l **视频结构冲突-叙述-金句**可保证播放量突破5000（无违规情况下）。

l **剪辑比拍摄更重要** ，但专业拍摄建议有限（口播视频简化了拍摄流程）。

**2.抖音视频制作流程**

l **制作流程分为以下步骤** ：

o **选题与文案** ：在微信中撰写内容，通过聚易网检测敏感词（如政治相关或极限词）。

o **提词与拍摄** ：使用灵感提词器辅助，选择安静环境拍摄。

o **剪辑处理** ：

l **降噪与音量调整** ：突出人声。

l **剪掉气口** ：删除错误或过长停顿以加快节奏。

l **滤镜与字幕** ：选择明亮滤镜，避免敏感词。

l **配乐** ：固定风格以强化品牌印象。

l **时间成本优化** ：

o 初期全流程耗时45分钟（剪辑占30分钟）。

o 后期外包剪辑后缩短至15-30分钟，实现**低成本持续输出**。

l **抖音核心原则** ：**绝对低成本**与**内容吸引力**。

o 连续7天每天200元无效投放就放弃（需坚持）

o 忽略作品质量是核心，投流仅提升曝光概率

o 不理解"运气因素"在爆款中的客观存在

o

o 比视频审核更严格

**一、创业与金融实战课**

2018年曾提出金融创新观点并发表文章，虽当前数据存在偏差且行业环境变化，但核心思路仍具参考价值。

**1.票据收益权资产证券化**

1) 承兑汇票

l **承兑汇票**指承诺在特定期限内兑付的汇款文书，持有人可凭票面约定时间向付款人主张债权

l **分类** ：

o **银行承兑汇票（银承）** ：付款主体为银行

o **商业承兑汇票（商承）** ：付款主体为企业

l **行业规模** ：中国当前承兑汇票存量规模约十余万亿元

2) 承兑汇票的案例应用

l **票据生成场景** ：A公司（强势买方）向B公司采购钢材时签发1亿元承兑汇票，约定6个月后付款

l **票据流转逻辑** ：

l **背景起源** ：2013年前后中国金融创新时期出现，与"某某经济学"理论及其实践者（包括财经刘、发改杜、广东朱、山东郭、中信常等）密切相关

l **核心机制**：通过收益权资产证券化帮助企业融资，将应收账款等收益权打包成理财产品出售

1. 金融创新模式

l **交易所爆发**：全国各地同时出现金交所、票交所、股交所等国有/半国有交易平台

l **主要业务**：以企业应收账款为抵押物制作理财产品，收益率普遍在8%-15%区间

l **特殊现象**：这些高收益产品多作为福利内部消化，普通民众较少接触

2. 票据收益权融资机会

l **运作闭环**：大企业发债→ 国有交易所通道→理财产品→ 国企职工购买

l **政策转折** ：P2P被叫停后，发现地方隐性债务问题严重（包括融资平台、PPP、政府购买服务等）

l **风险焦点**：商业承兑汇票成为最不透明的债务形式，国家开始全面整顿

3. 地方债问题

l **监管措施**：直接叫停各类交易所发债业务，导致大量交易中心转型/关闭

l **债务困境**：原有永续债模式（借新还旧）难以为继，地方政府面临偿债压力

l **系统风险**：隐性债务规模难以统计，包括政府担保、PPP项目、专项基金等多种形式4. 高手动手解决债务问题

l **创新方案**：中企云链模式（央企+地方政府+银行合作）

l **运作方式**：将隐性债务显性化，通过银行发债融资

l **行业特点**：需要极强资源整合能力，市场容量巨大但只能解决部分问题

5. 普通人的机会

l

l **供需分析**：

o 融资方：国企/大型民企通过票据抵押融资（收益率8%-36%，风险可控区间<24%）

o 投资方：民众理财渠道狭窄（银行定存4%，股市高风险，房地产受限）

l **创新方向**：

o 资产证券化：将高收益票据拆分为小额投资份额

o 借鉴模式：参考股票、虚拟币、潮鞋等低门槛投资品的成功经验

o 目标人群：满足普通投资者 "钱生钱"需求（如2000元炒股、1000元炒鞋、50元炒盲盒）

o

n 总部出资5%-20%，地方投资者出资80%-95%

n 总部负责经营管理，地方投资者提供资源但不参与决策

o **风险分配**：

n 亏损时总部仅承担5%-20%损失

n 前期收益先扣除经营成本（总部可回收部分投入）

o **优势**：零成本快速扩张，开店速度显著提升

o

o **收购时机** ：1-3年后门店度过冷启动期

o **收购方式**：

n 经营良好：用总部股权增持至51%以上完成并表

n 经营不善：通过管理费已收回成本，维护舆论减少负面影响

o **估值基础**：引入主流投资机构（如1亿占股2%），使总部估值达50亿

o **最终目标** ：3-5年后100%控股优质门店

o **收购对价**：现金+股权组合（门店估值=年利润5-10倍）

o **资本运作**：

n 利用资本市场30-50倍溢价空间

n 上市前倾向用现金收购保留股权

o **规模效应** ：30家门店中需有7家控股51% ，3家全资以增强市场信心

o

o **正循环机制**：

n 门店越多 → 总部估值越高→投资者信心越强

n 总部专注品牌/技术，投资者提供本地资源

o **风险控制**：

n 只收购盈利门店规避新店风险

n 用股权为亏损门店兜底（如1200万亏损用等值总部股权补偿）

o **终极目标**：形成"局部融资→快速扩张→估值提升"的资本运作闭环

l

l **产品同质化**：酒水供应无壁垒，竞品快速涌现（如红杉资本投资的RISSE酒馆、 COMMUNE和WAT等），行业价值主要依赖资本催熟先发优势。

l

l **股权置换策略**：选择5-6个城市最赚钱的加盟商，用总部1%-2%股权置换其门店所有

权，在开曼群岛注册上市主体。黑蚁资本背书增强可信度，强调"卖股权比卖啤酒更赚钱" 的IPO预期。

l **门店分类管理**：优质门店转为直营店统一宣传，经营不善的门店保留为经销商，选址隐蔽区域避免商誉损失。

o 开业15天内买酒水送同款+66元无门槛券

o 微信群立减+节日福利（如情人节送玫瑰花）

l **资本叙事逻辑**：所有运营动作均强化"用户经营 "而非 "啤酒销售" 的定位，向资本市场展示数字化管理能力。核心是通过金融思维利用他人资金和概率盈利。

3）个人投资者招募

l
