# 新能源领域决策洞察引擎

## 概述
面向新能源领域商业咨询服务的智能决策支持引擎。输入客户在**产业链中的角色、所处场景、核心痛点**，系统从跨子领域案例库、可复用创新方法论库与关键政策库中，生成结构化诊断建议——匹配同场景标杆、提供跨领域方法论映射、可落地的政策窗口与市场准入检查、为上游设备供应商提供专属切入路径与场景适配度评估，并结合财务逻辑推演与风险边界输出，帮助咨询顾问为客户构建全场景、全产业视角的决策认知，降低试错成本，挖掘差异化增长可能性。

## 目标用户
- **主要用户**：新能源行业商业咨询顾问，在客户诊断、方案推演、会议准备阶段使用
- **间接受益者**：成长期新能源企业决策层（发电企业、设备供应商、解决方案集成商、投资方），通过顾问输出获得深度洞察

## 核心功能

### 功能一：角色×场景双维画像输入
顾问将客户按两层结构化录入：
1. **产业链角色模板**：设备供应商（BMS/储能/光伏/飞轮/液流/热泵/重卡换电/节能设备等） / 解决方案集成商 / 聚合商 / 投资方 / 终端用能企业 / 工业园区管理方
2. **目标场景**（可多选）：虚拟电厂聚合运营 / 工商业微电网 / 矿山离网供电 / 户用离网光伏 / 车网互动V2G / 零碳园区 / 调频辅助服务 / 重卡换电 / 集中式新能源报价 / 非洲新兴市场（赞比亚/肯尼亚/莫桑比克）/ 乌克兰战后重建 / 钢铁等高耗能工业节能
3. **核心痛点标签**：盈利模式模糊 / 市场准入障碍 / 技术路线选择 / 出海风险未知 / 资源匹配不足 / 政策窗口把握
- 直接支持的画像模板：skus/procedural/skill_001 评估同一集团在同一省份的新能源场站是否满足集中报价的身份与规模约束、skus/procedural/skill_003 进入乌克兰新能源市场前的准备步骤与注意事项，适用于计划开拓乌克兰市场的中国企业、skus/procedural/skill_049 当企业计划进入非洲离网户用光伏市场时，遵循此四步法快速建立本地分销网络、skus/procedural/skill_077 在赞比亚开发新能源项目（太阳能、储能等）的投资审批流程，适用于外资投资者按步骤向赞比亚政府相关机构提交申请

### 功能三：供应商专属切入路径生成 + 场景适配度评估
**当客户为上游设备供应商时**，系统根据产品特性输出2-3条从卖设备到嵌入生态的路径，并评估其产品在不同场景的适配度（核心技术参数 vs 场景需求强度的匹配矩阵）。

**覆盖的设备类型与路径**：
- **BMS供应商**：
  - 路径A：技术封装为服务——结合[chunk: 动态生命周期的“资产价值再定义”框架]，将BMS电池健康预测能力转化为V2G调度依据，切入车网互动（skus/factual/sku_155 以50度电池、350公里续航为例，行驶20万公里仅消耗600次循环，总寿命约2000次中剩余1400次可用于V2G的循环利用论证）
  - 路径B：金融化锁客——借鉴[chunk: 硬件锁定的“实物金融化”信用体系]，为工商业储能提供带有远程通断与按次计费的BMS，切入储能分期租赁（skus/procedural/skill_050 在离网光伏系统中实现按天付费的PAYGo技术闭环，通过移动支付、激活码和远程控制完成付费解锁流程）
